# IWLZ Theme

IWLZ Theme 是一款面向中文内容站的 WordPress 主题，界面与交互参考 Halo 2 的 Theme Joe3，并按 WordPress 的模板、菜单、小工具、评论和自定义器体系重新实现。

本项目不是 Joe3 官方移植版，也没有直接打包 Joe3 的主题源码或图片素材。

## 主要功能

- Joe3 风格的品牌导航、横向文章列表和 300px 信息侧栏
- 桌面、平板和移动端响应式布局
- 浅色/深色模式与系统偏好检测
- 移动端抽屉导航和二级菜单
- 全局搜索面板，支持 `Ctrl/Cmd + K`
- 作者信息、站点统计、热门文章和标签侧栏
- 文章阅读量、阅读时间、目录、相关文章和上一篇/下一篇
- 代码块语言标识与一键复制
- PrismJS 1.30.0 本地语法高亮（MIT）
- WordPress 原生评论、菜单、小工具、自定义 Logo 和特色图片
- 对 Halo 迁移内容中的结构化摘要与多余闭合标签进行展示层兼容

## 安装

1. 将 `iwlz-theme` 目录放入 WordPress 的 `wp-content/themes/`。
2. 在 WordPress 后台进入“外观 > 主题”并启用 IWLZ Theme。
3. 在“外观 > 菜单”中为“主导航菜单”分配菜单。未分配时，主题会自动显示首页和高频分类。
4. 在“外观 > 自定义”中配置 Logo、主题色和 Joe3 主题设置。

## 主题设置

“外观 > 自定义 > Joe3 主题设置”提供以下选项：

- 首页公告与公告链接
- 作者卡背景图
- 默认文章封面
- 页脚补充文字

浅色和深色主题色位于“外观 > 自定义 > 颜色”。默认值分别为 `#fb6c28` 和 `#9999ff`。

## 模板结构

- `template-parts/feed.php`：首页、归档和搜索共用列表布局
- `template-parts/post-card.php`：文章列表条目
- `single.php`：文章详情、相关文章、文章导航与评论
- `sidebar.php`：作者信息、目录、热门文章、标签和小工具
- `js/theme.js`：主题模式、搜索、抽屉、目录、代码复制和返回顶部
- `style.css`：主题元数据、设计变量和响应式样式

## 开发检查

```bash
find . -name '*.php' -print0 | xargs -0 -n1 php -l
node --check js/theme.js
```

建议至少检查 `1440x1000`、`390x844` 两种视口，以及首页、文章页和深色模式。
